# SEVERIN — Cocktail Bar 

> Сайт культового коктейль-бара и ночного клуба в самом сердце Москвы.  
> Неон, авторские коктейли и атмосфера, которая остаётся с вами.

---

## О проекте

SEVERIN — это цифровой портал легендарного заведения, где каждый вечер превращается в шоу в неоново-розовом и фиолетовом свете.  
Проект объединяет современный дизайн, интерактивное меню и полноценную базу данных для управления коктейлями, заказами и аналитикой.

Сайт позволяет гостям познакомиться с атмосферой клуба, изучить коктейльную карту, найти напиток по вкусу и забронировать столик — всё в едином стиле, отражающем характер SEVERIN.

---

## Функциональность

### Главная страница / О клубе
- Приветственный блок с описанием концепции
- Блок статистики: количество коктейлей, лет опыта, форматы работы
- Галерея с фотографиями интерьера: танцпол, барная стойка, лаунж-зона
- Полная адаптивность под мобильные устройства и десктопы

### Меню коктейлей
- Каталог с фильтрацией по категориям (классические, популярные, авторские, сезонные)
- Поиск по названию и ингредиентам
- Динамическое отображение количества найденных коктейлей
- Карточки коктейлей с составом, ценой и стильным оформлением
- Состояние «ничего не найдено» с понятной подсказкой

### База данных (PostgreSQL)
- Полноценное хранение коктейлей, категорий, ингредиентов
- Управление заказами и столами
- Система пользователей с ролями (администратор, бармен, официант)

## Установка и запуск

### Требования
- Node.js (версия 18 или выше)
- PostgreSQL (версия 14 или выше)

### Шаги для запуска

```bash
# Клонировать репозиторий
git clone https://github.com/popaleonchika2000/severin_bar.git

# Перейти в папку проекта
cd severin-bar

# Запустить проект в режиме разработки
npm run dev
